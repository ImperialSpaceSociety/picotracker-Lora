
PC Utilities for FP-ATR-LORA1

In this folder you can find some examples to send commands to the LoRa IoT tracker.
Open a serial connection with 115200 baud speed and 8N1 settings.
See UM2487 for details. You can download it from www.st.com

'Txt' folder: this folder contains text files (.txt) with the implemented commands, use these 
with your preferred serial port terminal

'Tera Term' folder: this folder contains macros files (.ttl) to be used with teraterm software

'FP-ATR-LORA1 commands.ptp': use this project with Docklight (http://www.docklight.de/)
